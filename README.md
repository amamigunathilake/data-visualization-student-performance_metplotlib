# Student Performance Data Visualization

This project uses Python, Pandas, and Matplotlib to analyze student performance data.

## Tools Used
- Python
- Pandas
- Matplotlib

## Objectives
- Visualize marks distribution
- Analyze attendance vs marks
- Compare average marks by gender
- Identify low performing students

## Visualizations
- Histogram of marks
- Scatter plot of attendance vs marks
- Bar chart of average marks by gender

## Insights
- Higher attendance generally leads to better marks
- Female students show slightly higher average scores
- Low attendance often results in poor performance

This project demonstrates basic exploratory data analysis and visualization skills.
